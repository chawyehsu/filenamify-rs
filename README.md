> Convert a string to a valid filename

## Install

```toml
[dependencies]
filenamify = "0.1"
```

## Examples

```rust
use filenamify::filenamify;
let safe_filename = filenamify("//foo/bar/file");
println!("{}", safe_filename); // Prints "_foo_bar_file"
```
